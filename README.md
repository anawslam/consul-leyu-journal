# Consul Leyu Journal

## 项目简介

这是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，仅作为静态页面的集合与托管使用。所有内容以 HTML 文件形式存放，便于直接访问或嵌入其他环境。

## 目录说明

```
consul-leyu-journal/
├── pages/          # 存放各独立 HTML 页面
├── assets/         # 静态资源（CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件为一个独立的归档页面，命名遵循时间或主题顺序。
- `assets/`：页面引用的样式、脚本及媒体文件，按需组织。

## 页面归档说明

本仓库的页面主要记录各类信息、笔记或资料整理，以 HTML 格式保存。页面内容不涉及具体商业推广或诱导访问，仅作为存档与展示用途。如需引用或分发，请遵守相关许可证规定。

## 维护说明

- 本仓库由维护者不定期更新，新增页面会放入 `pages/` 目录。
- 不建议直接修改已归档的页面，如有必要可提交 Issue 或 Pull Request 讨论。
- 所有页面均为静态内容，不依赖后端服务，可直接在浏览器中打开。
- 本仓库不提供任何形式的托管保证，页面内容可能随时间调整或移除。
